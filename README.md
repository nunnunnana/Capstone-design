## 주요기능

### 메인페이지

<img src="https://github.com/nunnunnana/Capstone-design/assets/99165741/4cad8467-340a-4ceb-b83d-3d6d73fd78c0.png" width="600" height="300"/>

- 홈페이지 진입 시 처음 보이는 화면으로 용인시 탄천 길을 전부 나타낸다.

<img src="https://github.com/nunnunnana/Capstone-design/assets/99165741/2e0ec39e-3506-4a24-ab23-be28ff96a312.png" width="1200" height="300"/>

- 구를 선택하면 선택된 구의 탄천 길이 나타나고 해당 구의 동을 표시한다.

<img src="https://github.com/nunnunnana/Capstone-design/assets/99165741/90c0d4d7-9188-4073-ace8-454f106b4004.png" width="600" height="300"/>

- 동을 선택하면 해당 동의 탄천 길을 표시하고 전체 길이의 중간을 계산해서 중심화면을 정한다.


### 추천로
<img src="https://github.com/nunnunnana/Capstone-design/assets/99165741/1a904852-a730-44bf-a278-a4bfe5e2e954.png" width="800" height="300"/>

- 지역과 유형, 시간을 선택하고 검색하면 해당하는 탄천 길이 표시되며 화살표 버튼을 통해 지도를 깔끔하게 볼 수 있다.

<img src="https://github.com/nunnunnana/Capstone-design/assets/99165741/7b564a23-bf85-438a-98a5-ac8440e6d293.png" width="600" height="300"/>

- 라인에 마우스를 올리면 탄천 이름이 표시되고 라인이 강조되며 클릭하면 이름, 거리, 시간이 나타난다.


### 길찾기
<img src="https://github.com/nunnunnana/Capstone-design/assets/99165741/64acba76-c321-4f38-a08c-09ded71b553d.png" width="800" height="300"/>

- 탄천 갈의 모든 진입로를 마커로 표시하고 마커를 클릭하면 이름과 버튼이 표시된다.

<img src="https://github.com/nunnunnana/Capstone-design/assets/99165741/734653ba-e440-4f6b-98b8-4586956dc525.png" width="800" height="300"/>

<img src="https://github.com/nunnunnana/Capstone-design/assets/99165741/12165c02-6548-485b-a2f5-50fef6b98975.png" width="600" height="300"/>

- 마커를 클릭해 출발과 도착을 설정하면 경로가 나타난다.
